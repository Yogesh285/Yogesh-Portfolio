# 🚀 Yogesh Portfolio Website

A modern, responsive, and high-performance personal portfolio website built with **React**, **Vite**, and **Tailwind CSS** to showcase my projects, technical skills, certifications, and professional journey as a **Cloud & DevOps Engineer**.

> 🌐 **Live Demo:** https://yogesh-portfolio-r59bvihly-yogesh-ingavale-s-projects.vercel.app/
---

## 📌 About

This portfolio highlights my experience, technical expertise, and hands-on projects in Cloud Computing and DevOps. It is designed with a clean user interface, smooth animations, and responsive layouts to provide an excellent user experience across all devices.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Yogesh285/Yogesh-Portfolio.git
```

### Navigate to the Project

```bash
cd Yogesh-Portfolio
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
